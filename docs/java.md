# Acquiring Java RE

## Ubuntu (apt-based distros)

Run:
```console
# apt-get update
# apt-get install default-jre
```

## Arch (pacman/aur distros)

Run:
```console
# pacman -Syu
# pacman -S jre-openjdk
```

## Windows

Download and install the latest JRE release from [download page](https://www.java.com/en/download/).
